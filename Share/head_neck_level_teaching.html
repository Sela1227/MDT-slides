頸部淋巴 Level
外科 I–VII 與 N 分期

頸部不是分「站」，是依固定的解剖界標分區。部分界標可由體表定位（舌骨、環狀軟骨、胸鎖乳突肌），較深的界標（顱底、副神經、莖突舌骨肌、頸動脈）則要靠手術所見或 CT／MRI 辨認。
而個管實務最容易錯的不是解剖，是常見黏膜頭頸癌的 N 分期有三套彼此不同的系統——用錯一套，期別就整個跑掉。

對象：癌症個案管理師 依據：AJCC 8th · 2013 Neck Level Consensus N 分期：鼻咽採 Version 9（2025/1/1 生效）；p16 陽性口咽採 Version 9（2026/1/1 生效）；口腔／喉／下咽仍為 第 8 版——讀報告請先確認版本 右頸側面觀，投影示意

03

常見黏膜頭頸癌最易混淆的三套 N 分期

本頁聚焦個管實務最常混淆的三套黏膜癌 N 系統：一般 p16 陰性頭頸癌、p16 陽性口咽癌、鼻咽癌。甲狀腺癌、唾液腺癌、頭頸皮膚癌及其他特殊病理另有各自的分期規則，不適用本頁。
同一顆 3 公分的單側淋巴結在三套下都是 N1；但一顆 5 公分的呢？答案完全不同。切換下方按鈕比較看看。（以下均為 AJCC 8 規則）

版本警示
AJCC 已由「版次（Edition）」改為逐疾病更新的「版本（Version）」，各部位的生效時間不同。頭頸部現況：
鼻咽 Version 92025/1/1 生效
HPV 相關（p16+）口咽 Version 92026/1/1 生效
唾液腺 Version 92026/1/1 生效（本頁未涵蓋）
口腔、p16− 口咽與下咽、喉 仍為 第 8 版尚未被 Version 9 取代
本頁三套 N 表均已對齊上表。兩個實務提醒：p16+ 口咽的 CAP Cancer Protocol 到 2026 年 3 月才發布，在此之前發出的病理報告可能仍引用第 8 版；而 AJCC 的疾病別對照表每年更新，使用前請確認最新狀態。
N定義
必記 01

先問部位，再問 p16，最後問版本

先確認原發部位，若是口咽再確認 p16 狀態，選對 N 系統，最後確認報告用的是第 8 版還是 Version 9——鼻咽自 2025/1/1、p16+ 口咽自 2026/1/1 已改用 V9，但口腔／喉／下咽仍是第 8 版。
AJCC 8 以 p16 免疫組織化學染色作為 HPV 相關口咽癌分期的替代指標；p16 與直接 HPV 檢測在檢驗學上並非完全同義。

必記 02

ENE 三套都算，但算法都不同

p16 陰性（AJCC 8）：臨床上明顯的 ENE(+) → cN3b。
p16 陽性口咽（V9）：任何明確的 iENE／pENE → N 上調一級。
鼻咽（V9）：只有進展型 ENE（明確侵犯肌肉／皮膚／神經血管）→ 直接 N3。
對照肺癌：ENE 只是風險因子，不進 N 分期。四者不可互推。

必記 03

p16+ 的 pN：顆數＋pENE

第 8 版只數顆數（≤4 為 pN1、>4 為 pN2）。Version 9 起要同時看 pENE：單顆無 pENE 是 pN1a、2–4 顆無 pENE 是 pN1b、1–4 顆併 pENE 直接跳 pN2。不看淋巴結大小。讀報告先確認它用的是哪一版。

術語陷阱

外科 VII vs 放療 VIIa／VIIb

傳統外科 neck-level 命名法的 Level VII 多指上縱膈氣管旁區。放療的 Grégoire 勾勒命名法裡，VIIa ＝ 咽後、VIIb ＝ 莖突後。兩套命名不可直接互換，看報告先確認是誰寫的。

個管實務對照

報告上看到什麼 → 代表什麼 → 你要跟進什麼。

報告寫法臨床意義個管跟進
p16 (+)
oropharynx		p16 陽性口咽癌，預後明顯較佳，分期系統與 p16 陰性完全不同。p16 IHC（至少中等強度、>70% 腫瘤細胞染色）仍是適當的替代指標，但六種情境建議加做 HPV 專一性檢測（如低盛行率地區、p16 染色模糊、染色與型態不符、跨部位大腫瘤、非扁桃／非舌根的口咽部位、臨床試驗）先確認報告用的是 AJCC 8 還是 Version 9（V9 自 2026/1/1 生效，CAP protocol 2026/3 才跟上）。V9 要求至少 6 顆淋巴結才能判 pN0；戒菸衛教（吸菸會抵銷 p16+ 的預後優勢）；此族群多為中年、在職，需協助工作與治療的排程
p16 與 HPV
檢測結果不一致		p16 與 HPV 專一性檢測不符者預後較差依文獻應以 HPV-independent（p16 陰性）口咽癌分期，不可套用 p16+ 系統。發現不一致時回報主治確認
iENE (+)
影像 ENE		p16 陽性口咽癌自 V9 起，明確的 iENE 使 cN 上調一級。iENE 定義為明確的影像徵象：腫瘤突破被膜進入結周脂肪或鄰近構造，或兩顆以上相鄰淋巴結融合成無法分割的單一結構iENE 不是 pENE 的代理指標（對 pENE 特異度高但敏感度僅中等），兩者是各自獨立的預後因子。分期改變不等於治療改變——勿向病人暗示治療會因此加重
Clinical／imaging
ENE (+)		p16 陰性黏膜癌的臨床 N：臨床上明顯的 ENE(+) 即 cN3b；M0 時通常屬 Stage IVB先確認這是臨床 ENE而非僅病理顯微 ENE，並核對原發部位與所用的分期系統
Pathologic
ENE (+)		術後高風險因子。pN 須依顆數、大小、側別與 ENE 規則判定，並非一律 pN3b（例如單顆同側 ≤3 cm 且 ENE(+) 為 pN2a）確認切緣、淋巴結顆數與 ENE 範圍。已切除的黏膜鱗癌若確認 ENE(+)，屬術後高風險，應由 MDT 評估術後同步化放療；不適合 cisplatin 者須個別討論替代策略或單純放療。提早安排營養管路評估與放療前牙科評估及必要處置
Selective neck
dissection I–III		選擇性廓清，通常用於口腔癌 cN0確認取樣 Level 是否涵蓋原發部位的高風險區。未接受過頸部治療的口腔黏膜鱗癌，完整廓清的 lymph-node yield 常以約 18 顆作為品質參考之一；明顯低於預期時應核對手術範圍、病理取材與既往治療，不宜單憑顆數判定廓清不足
Level VI
involved		中央區受侵犯，常見於甲狀腺癌、聲門下與下咽癌提醒注意喉返神經與副甲狀腺相關術後併發症：聲音沙啞、低血鈣
Level IIB
dissected		清掃到副神經後上方副神經損傷風險最高的區域。術後衛教肩部復健，早期介入可預防肩部下垂與慢性疼痛
Retropharyngeal
node (+)		咽後淋巴結，通常無法由常規頸部觸診發現主要依 CT、MRI 或 PET-CT 評估——確認影像已完成。陽性時會影響 nodal GTV／CTV、側別處理與劑量配置（部分病例的 elective CTV 原本就涵蓋咽後區）
